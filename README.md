# React Registration Form

A controlled multi-field registration form built with React, covering the common patterns of form handling.

## Features

- Fully controlled inputs backed by a single state object
- Fields: first/last name, email, password + confirmation, date of birth, gender, country
- Show/hide password toggles
- Profile picture file upload
- Terms-and-conditions acceptance before submit

## Getting started

```bash
git clone https://github.com/Ayinde225/react-registration-form.git
cd react-registration-form
npm install
npm start
```

Then open http://localhost:3000 in your browser.

## Tech stack

React, JavaScript, Controlled Components, Create React App
